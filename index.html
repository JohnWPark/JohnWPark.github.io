HelpingHand


HelpingHand is a crowdsourced app that offers assistance to users in exchange for a fee. Users can post a job and browse for jobs they want to perform, all while communicating in real time with other users to discuss details of a job.

  • Built With: AngularJS, Node.js, Firebase, Bootstrap
  • Deployed Project: Firebase
  • Github Repo: Helping Hand

Paws


This Ruby on Rails app is a crowdsourced service where pet owners can place the care of their four-legged counterparts in the trusted hands of other pet owners.

  • Built With: Ruby on Rails, Bootstrap
  • Deployed Project: Heroku
  • Github Repo: Paws

BunkBnB


BunkBnB is a crowdsourced app that is modeled after AirBnB. It allows users to find accommodations by inputting a variety of parameters to meet their transient housing needs. Users also have the option of being hosts and offering their rooms or homes to other BunkBnb users.

  • Built With: Ruby on Rails, JQuery, AJAX, Bootstrap
  • Deployed Project: Heroku
  • Github Repo: BunkBnB

PopFlix


PopFlix is a movie review app created in Ruby on Rails. It allows users to login and post images of their favorite movies as well as leave reviews for other movies that they discover using the search function.

  • Deployed Project: Heroku
  • Github Repo: PopFlix
